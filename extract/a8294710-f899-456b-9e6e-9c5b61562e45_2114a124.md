## Configuration

The following sections define the entries required in the configuration files to provide the set-up described above.

## SLEE configuration file

The SLEE configuration file should have the following entries:

# Set application up
APPLICATION=CCS ccsApp ./bin 1 1

# Define a service for each Application Handle
SERVICE=ccsServ 1 CCS ccsHandle
# Define service Keys
SERVICEKEY=INTEGER 0x50d0000008 ccsServ

# Define VSSP and SS7 interfaces
INTERFACE=SS7 sinapTcapInterface ./bin EVENT

## SLEE TCAP interface configuration file

The SLEE TCAP Interface configuration file should have the following entries:

PROTO=inap

SLEEKEY=5

SSNS=13

The following sections define the entries required in the configuration files to provide the set-up described above. Here we use two startup scripts specified in SLEE.cfg to configure the two interfaces.

## SLEE configuration file

The SLEE configuration file should have the following entries:

# Set relay application up
APPLICATION=TCRapp tcRelayApp ../bin 1 1

# Define a service for each Application Handle
SERVICE=Serv1 1 TCRapp Scp2Handle

# Define service Key to route INAP messages of service
# key 5 on SSN 13 and service key 5 on SSN 112
SERVICEKEY=INTEGER 0x10d00000005 Serv1
SERVICEKEY=INTEGER 0x17000000005 Serv1

# Define both SINAP SS7 interfaces
INTERFACE=SinapIFA sinapTcapInterfaceA.sh ../bin EVENT
INTERFACE=SinapIFB sinapTcapInterfaceB.sh ../bin EVENT

The following sections define the entries required in the configuration files to provide the set-up described above.

## SLEE configuration file

The SLEE configuration file for SCP1 should have the following entries:

# Set application up
APPLICATION=TCRapp tcRelayApp ../bin 1 1

# Define a service for each Application Handle
SERVICE=Serv1 1 TCRapp Scp2Handle

# Define service Key to route INAP
# messages of service key 5 on SSN 13
# and service key 5 on SSN 112
SERVICEKEY=INTEGER 0x10d00000005 Serv1
SERVICEKEY=INTEGER 0x17000000005 Serv1

# Define both TCAP interfaces
INTERFACE=SinapIFA sinapTcapInterfaceA.sh ../bin EVENT
INTERFACE=SinapIFB tcpTcapInterfaceB.sh ../bin EVENT

The sinapTcapInterfaceA.sh SLEE TCAP Interface configuration script (see section) should have the following form:

#!/bin/sh
exec sinapTcapInterface -ssns 112 -proto inap

The tcpTcapInterfaceB.sh SLEE TCAP Interface configuration script (see section) should have the following form:

#!/bin/sh

exec tcpTcapInterface -ssns 13 -proto inap -pc 100

# Set application up
APPLICATION=TCRapp tcRelayApp ../bin 1 1

# Define a service for each Application Handle
SERVICE=TCServ 1 TCRapp Handle1
SERVICE=AcsServ 1 Acs Handle2
# Define service Key to route INAP
# messages of service key 5 on SSN 13
# and service key 5 on SSN 112
SERVICEKEY=INTEGER 0x10d00000005 AcsServ
SERVICEKEY=INTEGER 0x17000000005 TCServ
# Define all TCAP interfaces
INTERFACE=SinapIFA sinapTcapInterfaceA.sh ../bin EVENT
INTERFACE=SinapIFB tcpTcapInterfaceB.sh ../bin EVENT
INTERFACE=SinapIFC tcpTcapInterfaceC.sh ../bin EVENT
